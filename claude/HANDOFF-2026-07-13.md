## Architecture (quick)

- **`src/App.jsx`** — single-file React app (~13k lines). All UI.
- **`netlify/functions/db.js`** — auth + storage (Supabase `app_stores`: tenant, key, value jsonb).
- **`netlify/functions/guardian-nightly.mjs`** — nightly scheduled: integrity watchdog + customer-setup
  QA + full backup (emailed + Google Drive).
- Global (shared-across-logins) stores use `usePersist("<key>")` in `AppInner` — e.g. `featureFlags`,
  `publicBrand`. Per-user data lives in `settings`.
- Data guards in db.js: wipe-guard, F19 severe-shrink guard, F20 rate-card protection, tombstones
  (`deletedClients` + `deleteCustomer` action), auto-snapshots + Backups & Restore admin panel.
- Styled dialogs: `uiConfirm/uiPrompt/uiAlert` (promise-based) + one `<DialogHost/>` mounted in `App`.
  **There are no native `window.confirm/prompt/alert` left — use the `ui*` helpers.**
