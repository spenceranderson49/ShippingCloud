## Session watch-outs (all hit today — don't repeat)
- **JSX comment trap ×2**: `{/* */}` before a JSX root inside `return (` or a ternary branch
  breaks esbuild. Use plain `/* */` there.
- **Pipe masking**: `npm run build | tail` hides failures (pipeline exit = tail's). Use
  `set -o pipefail` before any build&&push chain. Two broken builds got pushed before this was
  caught (Netlify kept serving the last good deploy; no user impact).
- **Temp screenshot builds**: pattern = sed a `window.__scEnterDemo` hook into App.jsx (the
  retail landing has no demo door anymore), build, copy bundle to scratchpad dist-live, serve
  locally, then `git checkout -- src/App.jsx`. **That checkout eats any uncommitted real
  edits** — it silently reverted the isolate fix + tab sizing once (re-applied in 710f3e6).
  Commit real work BEFORE temp-build cycles. Never commit the hook.
- **Stacking-context bug**: the rail panel is an absolute `-z-10` span; without `isolate` on the
  aside it paints BEHIND the page background ("bro the line is gone??").
- A container hiccup once dropped a local commit → push rejected; **fetch + rebase, never force**.
- To open the demo's orders rail in automation: set localStorage `sc_u/demo/ship.ordersOpen`=true
  and reload (clicking the collapsed tab was flaky).
